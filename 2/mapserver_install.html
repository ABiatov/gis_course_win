![][02-mapserver-01] Установка Web-сервера Apache ---------------------------- Для установки Web-сервера Apache запустите инсталлятор OSGeo4W и установите пакет *Web &#10139; apache*. Чтобы убедиться в том, что Apache установился корректно, перейдите в браузере по адресу `http://IP-адрес Web-сервера`. Дожна отобразиться страница следующего содержания: ![][02-mapserver-02] Установка MapServer ------------------- Аналогично, используя инсталлятор OSGeo4W, установите пакет *Web &#10139; mapserver-dev*. Чтобы убедиться в том, что MapServer установился корректно, перейдите в браузере по адресу `http://IP-адрес Web-сервера/cgi-bin/mapserv`. Должна отобразиться страница, содержащая следующий текст: No query information to decode. QUERY_STRING is set, but empty.
