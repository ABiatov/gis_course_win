**ГИС для разработчиков** - это курс для разработчиков ПО, рассказывающий о том, как встроить ГИС-функциональность в собственные настольные приложения. **Пройдя этот курс вы узнаете:** * О проекциях, системах координат и о том, как производить пересчет данных из одной системы координат в другую * О различных форматах геоданных * О библиотеках и утилитах командной строки, предназначенных для обработки геоданных * О расширении языка SQL для обработки геоданных * Об использовании геоинформационных систем с открытым кодом для создания собственных ГИС приложений на их основе * Многое другое **Предполагается, что вы уже имеете:** * представление о работе в командной строке * опыт создания простейших SQL запросах * навыки создания програмного кода (на языках C/C++) **Для начала работы вам понадобится:** * Развернутый дистрибутив Windows или Linux * Установленный QGIS 2.x и новее * Установленная GRASS GIS 7.x * Установленные утилиты GDAL * Установленные утилиты PROJ4 Этот курс является базой для обучения созданию ГИС, проводимом компанией NextGIS. В ГИС области все довольно быстро меняется и этот курс находится в постоянной разработке. Присылайте нам замечания [почтой](http://nextgis.ru/contact/).
